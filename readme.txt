== Description ==

* CZ: Zobrazuje datum a české svátky (jmeniny) v postranním panelu (možno použít jako widget).
* EN: Shows actual date and the czech name days in the sidebar (possible as widget).

== Installation ==

1. Upload the plugin to your blog, or install via plugin management.
1. Activate it. 
1. Add the widget to your sidebar.
1. You're done!
